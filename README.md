## Installation

```powershell
git clone https://github.com/bissakov/nvim-portable.git

Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
.\install.ps1
.\start-shell.bat
goneovim
```

```powershell
git clone https://github.com/bissakov/nvim-portable.git

powershell -NoProfile -ExecutionPolicy Bypass -File .\install.ps1
.\start-shell.bat
goneovim
```
